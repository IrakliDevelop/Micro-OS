# sysinfo

## BRIEF
Display system information

## USAGE
sysinfo

## DESCRIPTION
Displays detailed information about the browser, device, and system running the terminal.

## DETAILED
The sysinfo command shows comprehensive information about your system and browser environment.

**INFORMATION DISPLAYED:**

Browser Details:
- Browser name and version
- User agent string
- Platform (OS)
- Language settings

Screen Information:
- Screen resolution
- Color depth
- Current viewport size

Device Information:
- CPU cores (if available)
- Memory (if available)
- Online/offline status
- Network connection type (if available)

**USE CASES:**
- Debugging browser-specific issues
- Checking system capabilities
- Verifying device specifications
- Testing responsive design breakpoints

**PRIVACY NOTE:**
All information is read from standard browser APIs. Nothing is sent to any server - this is purely local.

## EXAMPLES
```
sysinfo
```
