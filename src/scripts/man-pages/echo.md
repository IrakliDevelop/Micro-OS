# echo

## BRIEF
Echo text to the terminal

## USAGE
echo <text>

## DESCRIPTION
Prints the provided text to the terminal. Useful for displaying messages or testing output.

## DETAILED
The echo command displays text in the terminal output.

**BEHAVIOR:**
- Prints all arguments separated by spaces
- Does NOT interpret special characters or escape sequences
- Output appears as plain text

**COMMON USES:**
- Testing the terminal
- Displaying messages
- Debugging command strings
- Simple text output

The echo command is simple and straightforward - what you type is what you see.

## EXAMPLES
```
echo Hello, World!
echo Testing 123
echo Multiple words are joined
```
