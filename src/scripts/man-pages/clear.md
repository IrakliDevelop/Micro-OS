# clear

## BRIEF
Clear the terminal screen

## USAGE
clear

## DESCRIPTION
Clears all output from the terminal screen, giving you a fresh view. Command history is preserved.

## DETAILED
The clear command removes all visible output from the terminal, providing a clean slate.

**BEHAVIOR:**
- Removes all terminal output lines
- Does NOT clear command history (use arrow keys to access)
- Does NOT affect any files or settings
- Instant operation, no confirmation needed

**WHEN TO USE:**
- Terminal is cluttered with old output
- You want to focus on new tasks
- Before taking a screenshot
- When output is too long to scroll through

**TIP:**
After clearing, you can still access your command history with the up/down arrow keys.

## EXAMPLES
```
clear
```
