# boot

## BRIEF
Replay the boot sequence

## USAGE
boot

## DESCRIPTION
Replays the animated terminal boot sequence that appears when you first open the terminal.

## DETAILED
The boot command re-runs the startup animation sequence.

**WHAT IT DOES:**
- Clears the terminal
- Shows the system initialization messages
- Displays the welcome screen
- Runs the same animation you saw on first load

**USE CASES:**
- Showing the terminal to someone new
- Nostalgic recreation of the startup
- Clearing terminal in a stylish way
- Testing the boot sequence after modifications

**THE BOOT SEQUENCE INCLUDES:**
- System initialization messages
- Component loading simulation
- Welcome message with ASCII art
- Ready prompt

Note: This is a cosmetic command and doesn't actually restart or reset the terminal.

## EXAMPLES
```
boot
```
