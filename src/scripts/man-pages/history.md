# history

## BRIEF
Show command history

## USAGE
history

## DESCRIPTION
Displays a numbered list of all commands you have entered in the current session.

## DETAILED
The history command shows all commands you've executed since the terminal started.

**OUTPUT FORMAT:**
Each command is shown with:
- Line number (sequential, starting from 1)
- The full command as you typed it

**HISTORY NAVIGATION:**
You can navigate history without using this command:
- Arrow Up: Previous command
- Arrow Down: Next command

**PERSISTENCE:**
Command history is stored in memory only. It's cleared when you:
- Close the browser tab
- Refresh the page
- Clear browser data

The history is NOT stored across sessions.

## EXAMPLES
```
history
```
